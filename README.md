# Teslo API

1. Clone project

2. run `yarn install`

3. Clone file `.env.example` and rename to `.env`

4. Change environment data

5. Run DB

```
docker-compose up -d
```

6. Start Project `yarn start:dev`

7. Run SEED

```
localhost:3000/api/seed
```
